🔑 YouTube API 키

YouTube Data API v3 키를 입력하세요.
Google Cloud Console에서 발급받을 수 있습니다.

⚙️ 프록시 서버 설정 (Apps Script)

Google Apps Script 웹 앱 URL을 입력하세요. 구글 서버를 통해 영상 주소를 가져와 차단을 우회합니다.

📋 배포 방법:
1. script.google.com 접속 → 새 프로젝트
2. Code.gs 파일 내용 전부 복붙
3. 배포 → 새 배포 → 웹 앱 (반드시 모든 사용자 설정)
4. 생성된 웹 앱 URL을 여기에 입력
영상 로딩 중…
